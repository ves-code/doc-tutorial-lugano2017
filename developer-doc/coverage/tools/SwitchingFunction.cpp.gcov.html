      36             : /*
      37             : Functions that measure whether values are less than a certain quantity.
      38             : 
      39             : Switching functions \f$s(r)\f$ take a minimum of one input parameter \f$d_0\f$.
      40             : For \f$r \le d_0 \quad s(r)=1.0\f$ while for \f$r > d_0\f$ the function decays smoothly to 0.
      41             : The various switching functions available in plumed differ in terms of how this decay is performed.
      42             : 
      43             : Where there is an accepted convention in the literature (e.g. \ref COORDINATION) on the form of the 
      44             : switching function we use the convention as the default.  However, the flexibility to use different
      45             : switching functions is always present generally through a single keyword. This keyword generally 
      46             : takes an input with the following form:
      47             : 
      48             : \verbatim
      49             : KEYWORD={TYPE <list of parameters>}
      50             : \endverbatim  
      51             : 
      52             : The following table contains a list of the various switching functions that are available in plumed 2
      53             : together with an example input.
      54             : 
      55             : <table align=center frame=void width=95%% cellpadding=5%%>
      56             : <tr> 
      57             : <td> TYPE </td> <td> FUNCTION </td> <td> EXAMPLE INPUT </td> <td> DEFAULT PARAMETERS </td>
      58             : </tr> <tr> <td>RATIONAL </td> <td>

      60             : s(r)=\frac{ 1 - \left(\frac{ r - d_0 }{ r_0 }\right)^{n} }{ 1 - \left(\frac{ r - d_0 }{ r_0 }\right)^{m} }
