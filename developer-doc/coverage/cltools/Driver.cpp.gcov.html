     104             : The following command tells plumed to postprocess the trajectory contained in trajectory.xyz

     106             : stride keyword is given a stride equal to \f$n\f$ then it will be performed only on every \f$n\f$th
     107             : frame in the trajectory file.
     108             : \verbatim
     109             : plumed driver --plumed plumed.dat --ixyz trajectory.xyz
     110             : \endverbatim
     111             : 
     112             : The following command tells plumed to postprocess the trajectory contained in trajectory.xyz.
     113             :  by performing the actions described in the input file plumed.dat. Here though
     114             : --trajectory-stride is set equal to the frequency with which frames were output during the trajectory

     117             : had we run the calculation we are running with driver when the MD simulation was running.

     119             : plumed driver --plumed plumed.dat --ixyz trajectory.xyz --trajectory-stride 100 --timestep 0.001
     120             : \endverbatim
